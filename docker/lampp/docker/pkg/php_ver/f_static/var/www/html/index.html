Example of apache2 virtual hosts:
Edit your /etc/hosts
    127.0.0.1 php56.lan php74.lan php81.lan

To use your own www data:
1) create file in current directory
    www/enabled/php74.lan/public_html/index.html
        hello world!

2) run docker with --volume option
    docker run \
        --publish 10022:22 \
        --publish 80:80 \
        --publish 3306:3306 \
        --publish 5432:5432 \
        --volume ${PWD}/mnt/www:/var/www
3) go page
    http://localhost
